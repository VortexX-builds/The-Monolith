# Code Review Skill — Sloak Project

## Before Marking Any Component Complete, Check:

### Functionality
- [ ] All links go to correct routes
- [ ] Navbar dropdown works on mobile
- [ ] Lightbox opens, closes, prev/next works
- [ ] Contact form mailto: opens correctly with fields pre-populated
- [ ] Preloader fires only on first load

### Design
- [ ] Uses CSS variables for all colors — no hardcoded hex values
- [ ] Correct fonts loaded (Barlow Condensed headings, DM Sans body)
- [ ] Red used sparingly as accent only
- [ ] No decorative animations added

### Performance
- [ ] All images have loading="lazy"
- [ ] All page components are lazy-loaded in App.jsx
- [ ] No unused imports

### Content
- [ ] No Lorem Ipsum
- [ ] "Consultancy" spelled correctly everywhere (not "Constancy")
- [ ] "Maintenance" spelled correctly everywhere (not "Maintanance")
- [ ] All contact details match brief exactly

### Code Quality
- [ ] No inline styles
- [ ] No console.log statements
- [ ] Component names match file names
- [ ] Props are typed or documented with comments if ambiguous
